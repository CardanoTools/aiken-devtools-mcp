# Aiken Documentation

[](#aiken-documentation)

This is a website powered by [nextra](https://nextra.site/). It contains all of [Aiken's](https://github.com/aiken-lang/aiken) docs and is available at [https://aiken-lang.org](https://aiken-lang.org).

## Dev

[](#dev)

This project uses [pnpm](https://pnpm.io/).

When working locally run these commands:

pnpm install

pnpm dev

The site should now be available at [http://localhost:3000](http://localhost:3000).
